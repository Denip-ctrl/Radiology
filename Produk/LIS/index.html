Sistem Informasi Radiology

Sistem Informasi Manajemen yang terkomputerisasi, yang didesain untuk menangkap, menyimpan, dan menampilkan gambar medis

Fitur

  • Supports DICOM ver. 3
  • Vendor-Neutral Archive
  • Supports multi modality
  • Supports Mozilla Firefox, Chrome and Safari browser
  • Available as Android app
  • Security: SSL, Role Based Authorization, database encryption.
  • System integration ready: HIS, RIS, PACS, EMR using DICOM, HL-7 dan Restful API protocols.
  • User friendly with simple intuitive design
  • Access to patient radiology record history
  • Customizable workflow and report
  • Pure Web HTML5 based viewer, that is accessible anywhere that connected to internet
  • Centralized process and storing of the images
